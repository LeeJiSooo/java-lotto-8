# java-lotto-precourse
# 3주 차 로또 미션

## 구현할 기능 목록

### 1. 입력 및 예외 처리
- [ ] 로또 구입 금액 입력 (`1,000`원 단위, 숫자 검증)
- [ ] 당첨 번호 6개 입력 (쉼표 구분, 숫자/범위/중복 검증)
- [ ] 보너스 번호 1개 입력 (숫자/범위/당첨 번호와 중복 검증)
- [ ] 예외 발생 시 `[ERROR]` 메시지 출력 후 해당 입력 재시도

### 2. 도메인 로직
- [ ] `Lotto` 클래스: 번호 6개 유효성 검사 (개수, 중복, 1~45 범위) 및 오름차순 정렬
- [ ] 로또 발행: 구입 금액만큼 `Lotto` 객체 목록 생성 (`Randoms` API 사용)
- [ ] `Rank` Enum: 1~5등, 꽝 정보(상금, 일치 조건) 관리 (else/switch 없이)
- [ ] `WinningLotto` 클래스: 당첨 번호(Lotto)와 보너스 번호 캡슐화 (보너스 번호 중복 검증)
- [ ] 당첨 로직: 구매한 로또 목록과 `WinningLotto`를 비교하여 `Rank` 결정
- [ ] 통계 계산: 당첨 등수별 개수 집계 및 총 수익률 계산 (소수점 첫째 자리 반올림)

### 3. 출력
- [ ] 구매한 로또 개수 및 모든 로또 번호(오름차순) 출력
- [ ] 당첨 통계 (3개 일치 ~ 6개 일치) 형식에 맞게 출력
- [ ] 총 수익률(소수점 첫째 자리) 출력

### 4. 테스트
- [ ] `Lotto` 클래스 유효성 검사(개수, 중복, 범위) 테스트
- [ ] `WinningLotto` 보너스 번호 중복 검사 테스트
- [ ] `Rank` Enum 당첨/꽝 판별 로직 테스트
- [ ] 수익률 계산 로직 테스트
